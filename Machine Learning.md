##### 正规方程![image-20210430174937134](http://img.ron.zone/20210527232516.png)

如果X ^ TX是不可逆的，常见的原因可能是:

- 冗余特征，即两个特征密切相关(即线性相关)

- 太多的特性(例如m≤n)。在这种情况下，删除一些特性或使用“正则化”

##### 正规方程和梯度下降比较

| 梯度下降                   | 正规方程需要                                            |
| :------------------------- | :------------------------------------------------------ |
| 需要选择确定学习速率α      | 不需要选择确定学习速率α                        |
| 需要多次迭代                           | 不需要迭代                                      |
| O (kn^2)            | O (n^3), 需要计算 X^TX的逆矩阵 |
| n 非常大时任然适用 | n太大计算缓慢                      |
